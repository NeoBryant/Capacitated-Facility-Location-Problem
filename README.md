# Capacitated-Facility-Location-Problem
# 固定容量设备选址问题


## 文件夹结构
```
├─data //测试数据 
│
└─src //源代码
    │  greedy_algorithm.py //贪心算法策略1
    │  greedy_algorithm2.py //贪心算法策略2
    │  load_data.py // 文件读取，数据加载
    │  local_search_algorithm.py //局部搜索算法
    └─ main.py //测试程序
```

## 程序运行
使用下面命令编译：
```
python main.py
```
需要在main.py文件中修改需要调用的算法函数，默认为贪心算法策略1
